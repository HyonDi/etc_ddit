웹 어플리케이션 구성
	- 화면
	- 데이터베이스
	- 서버

	화면
		- HTML : 화면에 그림을 보여주는 역할
		- CSS : 그림을 예쁘게 꾸며주는 역할
		- Javascript : 그림이 동작하게 만드는 역할

	데이터베이스
		- Oracle : 데이터를 저장하고 필요한 데이터를 제공하는 역할

	서버
		- Java : 데이터를 옮겨주는 역할

Java의 구성
	- 2장 : 변수
	- 3장 : 연산자
	- 4장 : 조건문과 반복문
	- 5장 : 배열
	- 6장 : 객체지향1(클래스)
	- 7장 : 객체지향2(상속)
	- 8장 : 예외처리
	- 9장~ : 자바에서 제공하는 부가적인 기능들
