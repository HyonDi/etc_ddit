
정규식 테스트

아이디 : 영문소문자로 시작, 숫자와 조합(4~12글자)
=> 영문자만 입력해본다.
=> 숫자로만 입력해본다.
=> 숫자로 시작해서 영문자로 끝나게 입력해본다.
=> 영문대문자로 시작해본다.
=> 한글로 입력해본다.
=> 4글자미만/13글자이상으로 입력해본다.
=> 공백으로 입력해본다.


이름 : 한글로만 구성 또는 영문으로만 구성.
=> 한글과 영문을 혼합해서 입력해본다.
=> 한글로 시작하고 숫자나 영문으로 끝나게 입력해본다.
=> 영문으로 시작하고 한글이나 숫자로 끝나도록 입력해본다.
=> 숫자로 입력해본다.
=> 특수문자를 사용해본다.
=> 한글 2글자미만/ 6글자 이상으로 입력해본다.
=> 영어 4글자미만/13글자 이상으로 입력해본다.
=> 공백으로 입력해본다.

비밀번호 :영문소문자대문자 숫자 특수문자 하나이상씩 조합.
=> 영문소문자로만 입력해본다.
=> 영문 대문자로만 입력해본다.
=> 숫자로만 입력해본다.
=> 특수문자로만 입력해본다.
=> 영문소문자와 숫자
=> 공백으로 입력해본다.

이메일 : 
=> @ 없이 입력해본다.
=> . 없이 입력해본다.
=> 이메일 @뒤에 .이 3번나오도록 해본다.
=> 한글로 입력해본다.
=> @앞뒤로 다른 특수문자를 입력해본다.
=> 공백으로 입력해본다.

휴대폰번호 : 000-0000-0000
=> - 없이 입력해본다.
=> 자릿수에 맞지 않게 입력해본다.
=> 공백으로 입력해본다. 
=> 한글 또는 영문을 입력해본다.
=> -를 4개 사용해본다.
